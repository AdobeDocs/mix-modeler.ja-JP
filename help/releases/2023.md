# 2023 Mix Modeler リリースノート

## 2023年11月


| 機能 | 説明 | 一般公開（GA） |
|---|---|---|
| **CSV ボタンのダウンロード** | ダウンロード CSV ボタンは、[&#x200B; プランインサイト &#x200B;](../plans/build.md)および[&#x200B; モデルインサイト &#x200B;](../models/insights.md#model-insights)の一部として利用できるようになりました。 | 2023年11月16日（PT） |
| データ範囲の分類テーブルの&#x200B;**CPA列** | CPA列は、Plan Insightsの[&#x200B; データ範囲の分類](../plans/build.md) テーブルに追加されます。 | 2023年11月16日（PT） |
| **有料チャネル ROIの可視化を予測** | [予測された有料チャネル ROI](../plans/build.md)のビジュアライゼーションがプラン インサイトに追加されました。 | 2023年11月16日（PT） |
| **モデルの概要UIが更新されました** | [&#x200B; モデルの概要](../models/overview.md) UIは、プランの概要UIに合わせて更新されます。 | 2023年11月16日（PT） |
| **モデルの再評価** | 既存のモデル [&#128279;](../models/overview.md#rescore)を使用して新しいデータの リスコアリングをトリガーする機能を追加しました。 | 2023年11月16日（PT） |
| **貢献度の可視化を更新** | 日付および基本メディアのビジュアライゼーション [&#128279;](../models/insights.md#model-insights)による貢献度が、順序付き積み重ねグラフに更新されます。 | 2023年11月16日（PT） |

## 2023年10月

| 機能 | 説明 | 一般公開（GA） |
|---|---|---|
| **データ取り込みの概要** | [Experience Platformの新しい概要クラスを使用した要約データ &#x200B;](../ingest-data/overview.md) （ウォールドガーデンの支出データ、テレビのフライトスケジュールなど）の取り込み。 | 2023年10月4日（PT） |
| **マーケティング投資の増分効果を測定** | [&#x200B; ビジネス目標または戦術的なキャンペーン目標をまたいで、マーケティングの増分ROIと影響を](../dashboard/overview.md)理解します。 | 2023年10月4日（PT） |
| **マルチタッチアトリビューション（MTA）とマーケティングミックスモデリング（MMM）の転移学習** | MMMとMTA [&#x200B; モデル &#x200B;](../models/overview.md)の統合により、移行学習を通じて自信をもって意思決定を行うことができます。 | 2023年10月4日（PT） |
| **マーケティング投資シナリオプランニング** | 複数の予算[&#x200B; プラン &#x200B;](../plans/overview.md)を作成し、その効果を比較して、ビジネスに最適な意思決定を行います。 | 2023年10月4日（PT） |
