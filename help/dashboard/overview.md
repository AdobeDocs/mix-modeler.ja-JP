# 概要ダッシュボード


AdobeMix Modeler の概要ダッシュボードでは、様々な設定可能なウィジェットを使用して、調整済みのデータとプランの概要を確認できます。

概要にアクセスするには、 ![ホーム](../assets/icons/Home.svg) **[!UICONTROL Overview]** を [AdobeMix Modeler] インタフェースの左側のレールから選択します。

ここから、以下の詳細を取得できます。

* [調整済みデータ](harmonized-data.md)

* [プラン](plans.md)
